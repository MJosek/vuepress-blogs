## 开始

这是一份在本地构建项目的指导的例子。
要获取本地副本并且配置运行，你可以按照下面的示例步骤操作。

### 依赖

这只是一个列出软件依赖和安装方法的例子。
* npm
  ```sh
  npm install npm@latest -g
  ```

### 安装

_下面是一个指导你的受众如何安装和配置你的应用的例子。这个模板不需要任何外部依赖或服务。_

1. 在 [https://example.com](https://example.com) 获取一个免费的 API Key。
2. 克隆本仓库
   ```sh
   git clone https://github.com/MJosek/vuepress-blogs.git
   ```
3. 安装 NPM 包
   ```sh
   npm install
   ```
4. 启动你的项目
   ```sh
  npm run docs:dev
   ```
